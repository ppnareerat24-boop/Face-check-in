# ระบบเช็คชื่อด้วยใบหน้า (Face Check-in)

เว็บแอปเช็คชื่อด้วยการจดจำใบหน้า รันบนเครื่องตัวเอง (localhost)
ประมวลผลใบหน้า **ในเบราว์เซอร์ทั้งหมด** ด้วย [face-api.js](https://github.com/vladmandic/face-api)
ภาพจากกล้องไม่ถูกส่งออกไปยังเซิร์ฟเวอร์ใด ๆ และข้อมูลทั้งหมดเก็บไว้ใน
เบราว์เซอร์ (IndexedDB) ของเครื่องนี้เท่านั้น

## ความสามารถ
- 📷 เช็คชื่อด้วยใบหน้าแบบเรียลไทม์ (กล้องซ้าย / ชื่อ-สถานะ-เวลา ขวา ตามแบบ)
- 🔐 **หน้าต่างผู้ดูแลระบบ** มีหน้า login ป้องกัน (รหัสผ่านเก็บแบบ hash)
- 📝 **ลงทะเบียนครั้งแรก** (อยู่ในหน้าต่างผู้ดูแล) — เก็บตัวอย่างใบหน้า 5 ภาพต่อคน
- 🕘 **ประวัติเช็คชื่อย้อนหลัง** (อยู่ในหน้าต่างผู้ดูแล) — กรองตามวันที่/ชื่อ, ส่งออก CSV, ล้างประวัติ
- ✅ เช็คชื่อได้ครั้งเดียวต่อคนต่อวัน (กันเช็คซ้ำ)
- 🔌 ใช้งาน **offline** ได้ (โมเดลถูกเก็บไว้ในเครื่องที่โฟลเดอร์ `models/`)

## วิธีใช้งาน

### 1. เปิดเซิร์ฟเวอร์
```bash
npm start
```
หรือ
```bash
node server.js
```
แล้วเปิดเบราว์เซอร์ (Chrome หรือ Edge) ไปที่:
```
http://127.0.0.1:8000
```
> ต้องเปิดผ่าน `http://localhost` / `127.0.0.1` เท่านั้น เบราว์เซอร์ถึงจะอนุญาตให้ใช้กล้อง
> (เปิดไฟล์ `index.html` ตรง ๆ แบบ `file://` กล้องจะไม่ทำงาน)

เปลี่ยนพอร์ตได้ด้วย: `PORT=9000 node server.js`

### 2. เข้าหน้าผู้ดูแลระบบ (หน้าแยก + login)
1. กดปุ่ม **สำหรับผู้ดูแลระบบ** (มุมขวาล่าง) → เปิด **หน้า `admin.html` แยกต่างหาก**
2. **ครั้งแรก**: ระบบจะให้ตั้งชื่อผู้ใช้ (ค่าเริ่มต้น `admin`) + รหัสผ่าน (อย่างน้อย 4 ตัว)
3. **ครั้งต่อไป**: ใส่ชื่อผู้ใช้/รหัสผ่านเพื่อเข้าสู่ระบบ (ต้อง login ทุกครั้งที่เข้าหน้านี้)
4. ในหน้าผู้ดูแลมี 2 แท็บ: **ลงทะเบียนครั้งแรก** และ **ประวัติเช็คชื่อ**
   มีปุ่ม **← กลับหน้าเช็คชื่อ** และ **ออกจากระบบ** ที่หัวหน้า

### 3. ลงทะเบียนครั้งแรก (แท็บในหน้าผู้ดูแล)
1. กรอกชื่อ-นามสกุล
2. เลือก **ประเภท**: นักเรียน/อาจารย์ หรือ บุคคลภายนอก (ถ้าเป็นบุคคลภายนอกต้องกรอกเหตุผล)
3. ดู **พรีวิวกล้อง** ในหน้าให้เห็นใบหน้าชัด แล้วกด **เริ่มจับภาพใบหน้า** (เก็บ 5 ตัวอย่าง)
4. กด **บันทึก**

### 4. เช็คชื่อ
แค่ยืนหน้ากล้อง ระบบจะแสดงชื่อและสถานะ **เช็คชื่อแล้ว!** (สีเขียว) พร้อมวันเวลา
ถ้าวันนี้เช็คไปแล้วจะขึ้น *เช็คชื่อไปแล้ววันนี้*

### 5. ดูประวัติ
เปิดหน้าต่างผู้ดูแล ▸ แท็บ **ประวัติเช็คชื่อ** เพื่อดู/กรอง/ส่งออกประวัติย้อนหลัง

> **ลืมรหัสผ่านผู้ดูแล?** ล้างได้ที่ DevTools (F12) ▸ Application ▸ Local Storage ▸
> ลบคีย์ `faceAdminAuth` แล้วระบบจะให้ตั้งรหัสใหม่

## โครงสร้างโปรเจกต์
```
Face_scan/
├─ server.js               เซิร์ฟเวอร์ static (Node ล้วน ไม่มี dependency รันไทม์)
├─ index.html              หน้าเช็คชื่อ (เลย์เอาต์ตามแบบ)
├─ admin.html              หน้าผู้ดูแลระบบ (login + ลงทะเบียน + ประวัติ)
├─ css/style.css
├─ js/
│  ├─ checkin.js           ลอจิกหน้าเช็คชื่อ (ลูปจดจำใบหน้า + แสดงผล)
│  ├─ admin.js             ลอจิกหน้าผู้ดูแล (login / ลงทะเบียน / ประวัติ)
│  ├─ faceengine.js        โหลดโมเดล / กล้อง / จับคู่ใบหน้า (ใช้ร่วมกันสองหน้า)
│  ├─ db.js                เก็บข้อมูลใน IndexedDB (ใช้ร่วมกันสองหน้า)
│  └─ vendor/face-api.js   ไลบรารี face-api.js (เก็บไว้ในเครื่อง)
└─ models/                 น้ำหนักโมเดล (tiny detector + landmarks + recognition)
```

## หมายเหตุทางเทคนิค
- การจับคู่ใช้ระยะ Euclidean ของเวกเตอร์ใบหน้า 128 มิติ เกณฑ์ `0.5`
  (ปรับได้ที่ `MATCH_THRESHOLD` ใน `js/faceengine.js` — ค่าน้อย = เข้มงวดขึ้น)
- ข้อมูลผู้ใช้และประวัติอยู่ใน IndexedDB ฐานชื่อ `faceCheckinDB`
  (ลบได้จาก DevTools ▸ Application ▸ IndexedDB หากต้องการรีเซ็ตทั้งหมด)
- ต้องการความแม่นยำสูงขึ้น: เพิ่มจำนวนตัวอย่างตอนลงทะเบียน หรือเปลี่ยนไปใช้
  ตัวตรวจจับ `ssdMobilenetv1` (มีโมเดลให้ใน `node_modules/@vladmandic/face-api/model`)
